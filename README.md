# Responsive Personal Portfolio Website

A responsive personal portfolio website built using HTML, CSS, and JavaScript. It showcases projects, technical skills, and contact information with a clean and modern design.

## Features

* Responsive design for all devices
* Smooth navigation
* Projects and skills section
* Contact section with social links
* Interactive and modern UI

## Technologies Used

* HTML
* CSS
* JavaScript

## How to Run

1. Clone the repository
2. Open the project folder
3. Run `index.html` in your browser
